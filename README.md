# React Pricing Card UI

A clean and responsive pricing card interface built with React. This project focuses only on the user interface and visual layout.

## Features

- Modern pricing card layout
- Responsive design
- Reusable React components
- Static UI only
- No backend functionality
- No payment integration

## Tech Stack

- React
- HTML
- CSS
- JavaScript

## Getting Started

```bash
git clone https://github.com/your-username/react-pricing-card.git
cd react-pricing-card
npm install
npm start
